To make a call to the authentication service using AJAX, you will need the following call in your script: 

    function Authenticate(DataToSend) {
        var responsePromise = $http({
            method: 'POST',
            url: 'OttjohnAuth.svc/Login',
            data: DataToSend,
            async: false,
            headers: {
                'Content-Type': 'application/json'
            }
        });
        return responsePromise;
    }
    function Logout() {
        var responsePromise = $http({
            method: 'POST',
            url: 'OttjohnAuth.svc/Logout',
            data: null,
            async: false,
            headers: {
                'Content-Type': 'application/json'
            }
        });
        return responsePromise;
    }

This, of course, assumes that the service name is OttjohnAuth. To check authentication on the server side, you need the HttpContext object. This is going to be available for you under System.Web. Checking if a user is authenticated is simple: 

    HttpContext.Current.User.Identity.IsAuthenticated

To check if a user in in a role, you can use the following: 

    HttpContext.Current.User.IsInRole("ROLENAME")
